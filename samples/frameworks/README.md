# MRZ Scanner - Angular, React, and Vue

These samples demonstrate how to use the `MRZScanner` API from the [Dynamsoft MRZ Scanner SDK](https://www.dynamsoft.com/use-cases/mrz-scanner/) in Angular, React (with hooks), and Vue apps to scan an MRZ using the camera.

## Features

- Easy integration with pre-built UI
- Scans one MRZ at a time from video
- Displays both the captured image and formatted MRZ data
- **Angular**: Renders `MRZScanner` inside an Angular component
- **Vue**: Renders `MRZScanner` inside a Vue component
- **React**: Renders `MRZScanner` inside a React container

## Quick Start

> [!WARNING]
> MRZ Scanner JS requires a valid license key to operate. Please see the [licensing documentation](https://www.dynamsoft.com/mrz-scanner/docs/web/guides/mrz-scanner.html#license) to sign up for a free trial or a full license.

Navigate to the root directory of one of the framework sample.

### 1. Install Dependencies

```shell
npm install
```

### 2. Build and Serve the Application

```shell
npm run dev
```

Then open the URL presented in the console to view the sample app.

### 3. Build for Production

```shell
npm run build
```

You can preview the production build for React and Vue by additionally running:

```shell
npm run preview
```
